# City Runners

Monorepo for a realtime, location based hide and seek gamed, centered around Hong Kong's MTR system.

## API
A Rust API connected to a realtime database using:
- Axum
- Tokio
- Websockets
- PostgreSQL

## Web
A frontend interface made with:
- Svelte + Sveltekit
- Axios & WS
- TailwindCSS + Shadcn/ui
